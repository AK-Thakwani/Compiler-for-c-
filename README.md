# C Compiler (Python)

## Overview
This project implements a mini C compiler in Python including:
- Lexical Analysis
- Syntax Parsing
- AST Construction
- Semantic Analysis
- Code Generation

## Architecture
Lexer → Parser → AST → Semantic Analyzer → Code Generator

## Features
- Variable declarations
- Arithmetic expressions
- Conditional statements
- Basic control flow
